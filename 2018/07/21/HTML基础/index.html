HTML基础

2018-07-21

一 HTML基础

  1. 浏览器

    • 浏览器是提供信息浏览服务的客户端
    • 浏览器内核(渲染引擎)解析
      • IE — trident
      • Chome /Opera — blink
      • Firefox — gecko
      • Safari — webkit
    • 浏览器内核: 也就是浏览器所采用的渲染引擎，渲染引擎决定了浏览器如何显示网页的内容，以及页面的格式信息

  2. 客户端（浏览器）与服务器之间的交互

    • 1.通过浏览器地址栏中的域名，到DNS服务器上（DNS解析/域名解析）找到对应服务器的外网IP
    • 2.通过外网IP找到对应的服务器
    • 3.通过地址栏中输入的端口号找到对应的服务，此外我们找到是相应端口号，而这个端口号会负责我们当前项目的处理
    • 4.告诉这个端口号我需要哪一个资源文件，这个端口号会把需要的“资源文件中的源代码”返回给客户
    • 5.客户拿到源码后，客户端的浏览器会把源码解析为相应的文件类型（例如：html文件，css样式文件，js文件···）

  3. HTML(超文本标记语言) — HyperText Markup Language

  4. Web标准

    • 结构标准 html

    • 表现标准 css

    • 行为标准 js

  5. 标签

    • 标签分类

      • 单双标签

        • 单标签 eg. <! DOCTYPE html>
          • 三个单标签
            • 1 注释标签
            • 2 分割线标签 hr
            • 3 换行标签 br
        • 双标签 eg.
          • 三个双标签
            • 1 标题 h1 - h6
            • 2 段落标签 p
            • 3 文本标签 font

        • 1 加粗
